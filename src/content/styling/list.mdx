# List Icons
Use icons as bullets for the list by adding class `bx-ul` to the unordered list element and include the icons inside the `li`

```html
<ul class='bx-ul'>
<li><i class='bx  bx-right-arrow'></i>List Item 1</li>
<li><i  class='bx  bx-x'></i>List Item 2</li>
<li><i  class='bx  bx-plus'></i>List Item 3</li>
</ul>
```
